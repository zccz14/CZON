# CZON - AI-Native Markdown 콘텐츠 엔진

[> 웹사이트 데모](https://czon.zccz14.com/)

AI 시대에 웹사이트 콘텐츠 크리에이터로서 우리는 더 스마트한 콘텐츠 제작 엔진을 가질 수 있습니다.

**콘텐츠로의 회귀: 최소한의 방해, 집중적인 작성**

문서가 본질로 돌아가고, 작성이 평온함으로 돌아가도록 합니다.

첫째, 저는 도서관 사서가 되고 싶지 않습니다. 때로는 어떤 내용을 적고 싶지만, 아직 제목을 정하지 못했거나 특별한 체계를 형성하지 못한 경우가 있습니다. LLM이 자동으로 제목을 생성하고, 요약을 만들며, 분류하고, 네비게이션 등의 정리 작업을 해주길 바랍니다.

둘째, 제 서재는 반드시 깨끗하게 청소되어야 합니다. 복잡한 구성 도구를 설정하고, 복잡한 문서 구성을 만지작거리며, 복잡한 구조를 좋아하지 않습니다. Docusaurus, VuePress, Astro와 같은 도구들의 구성 파일과 프로젝트 구조를 볼 때마다 머리가 아픕니다. 이제 CZON은 모든 구성을 .czon 디렉토리 아래로 숨기고 구석으로 쓸어넣어, 사용자가 작성 콘텐츠에만 집중할 수 있도록 하고 나머지는 CZON이 처리하도록 선택했습니다. 다른 디렉토리들은 모두 여러분의 작성 공간이며, Markdown 파일을 자유롭게 구성할 수 있습니다.

**모국어로의 회귀: 모국어로 작성, 부드러운 다국어 버전 구축**

인생은 짧아서, 저는 번역하기가 귀찮습니다. 사람은 모국어를 사용할 때 가장 자신의 사고 깊이를 발휘할 수 있습니다. 하지만 동시에 세계와 연결을 유지하는 것도 필요하며, 저는 다국어 소통을 포기하고 싶지 않습니다. 그래서 저는 모국어로 작성하면서도 더 많은 사람들이 제 콘텐츠를 볼 수 있도록 해주는 도구를 원했습니다. 현재의 i18n은 직접 번역해야 할 뿐만 아니라 콘텐츠 업데이트도 직접 유지보수해야 하니 너무 번거롭습니다. 저는 LLM을 사용하여 번역을 도와주도록 선택하여 많은 시간을 절약하고, 한 번의 클릭으로 여러 국가 언어 번역을 완료할 수 있습니다.

## 기본 기능

1. 🌍 **자동 다국어 번역**: AI를 사용한 증분 번역으로 사용자가 모국어로 Markdown을 작성하지만, 사용자는 다국어가 될 수 있습니다.
1. 💭 **자동 요약 추출**: AI를 사용하여 원본 텍스트의 내용을 분석하고 추출합니다.
1. 🧭 **스마트 분류 네비게이션**: AI를 사용하여 사이트 맵과 네비게이션을 생성하며, 소스 파일의 위치에 민감하지 않습니다.

## 정적 사이트 생성 (SSG)

CZON은 내장된 SSG 솔루션을 갖추고 있어, Markdown을 포함하는 임의의 git 폴더를 정적 HTML 사이트로 구축할 수 있습니다.

⚠️ 참고: CZON은 전문적인 정적 사이트 생성기(SSG)가 아니라, 콘텐츠 제작 및 관리에 중점을 둔 AI 기반 Markdown 콘텐츠 엔진입니다.

✅ CZON은 콘텐츠 제작 및 게시 과정을 단순화하여 사용자가 복잡한 구성과 도구에 방해받지 않고 작성 자체에 집중할 수 있도록 합니다.

❌ CZON은 복잡한 테마 사용자 정의 및 플러그인 생태계를 제공하지 않으며, AI를 통해 콘텐츠의 품질과 접근성을 향상시키는 데 집중합니다.

🔔 하지만, CZON은 여전히 정적 사이트를 생성할 수 있으며, 필요한 경우 CZON을 전처리기로 사용하여 다른 SSG 솔루션과 통합하여 개인화되고 아름다운 사이트를 생성할 수 있습니다.

## 빠른 시작

필수 조건:

- [Node.js](https://nodejs.org/) 설치됨 (버전 18 이상 권장, npx 명령어 필요)
- [OpenAI API Key](https://platform.openai.com/account/api-keys) 획득, 또는 OPENAI 호환 API Key
- Git 설치됨 (Git에서 Markdown 파일을 나열하고, .gitignore 규칙으로 필터링된 파일을 무시하기 위해)

먼저, 이미 git으로 관리되는 폴더에서 작업하세요:

환경 변수 설정

```bash
export OPENAI_API_KEY="sk-xxxxxx"  # 필수: 귀하의 OpenAI API Key로 교체하세요
export OPENAI_BASE_URL="https://api.openai.com/v1"  # 선택 사항, 기본값은 OpenAI 공식 API 사용
export OPENAI_MODEL="gpt-3.5-turbo"  # 선택 사항, 기본값은 gpt-3.5-turbo 모델 사용
```

사이트 구축, 다국어 번역 지원. 현재 디렉토리를 소스 디렉토리로 사용하여 `.czon/dist` 디렉토리에 출력합니다.

- `--lang` 매개변수를 여러 번 사용하여 생성할 언어 버전을 지정할 수 있습니다 (예: `--lang zh-Hans --lang en-US`).
- 소스 언어를 구성할 필요가 없으며, CZON이 자동으로 감지합니다.

```bash
npx czon@latest build --lang zh-Hans --lang en-US
```

**더 많은 매개변수 또는 도움말 보기**:

```bash
npx czon@latest
```

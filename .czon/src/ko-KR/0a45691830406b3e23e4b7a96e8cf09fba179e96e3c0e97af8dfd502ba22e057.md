# 사용 팁

## 본문에서 날짜 선언하기

본문에서 "오늘은 2026년 1월 8일입니다"와 같은 날짜 선언을 사용할 수 있습니다(형식 스타일은 자유롭습니다). CZON이 자동으로 인식하고 처리합니다.

기존 Markdown 문서는 YAML Frontmatter를 통해 날짜를 선언하며, 해당 필드를 기억해야 합니다. 그렇지 않으면 날짜가 올바르게 인식되지 않을 수 있습니다.

CZON은 본문에 직접 날짜를 선언하는 것을 지원하여 Frontmatter 필드 입력을 잊는 문제를 방지합니다.

AI가 본문의 날짜 정보를 자동으로 추출 및 추론하여 문서의 생성 날짜로 처리합니다.

이 방식은 더 유연하고 지능적이며, 사용자 경험을 향상시킵니다.

덧붙여, 날짜가 있는 문서는 타임라인 뷰를 생성하여 탐색과 관리가 용이합니다. 특히 일기, 블로그 등 타임라인이 필요한 콘텐츠 유형에 적합합니다.

# Mermaid 차트 테스트

이것은 CZON에서 Mermaid 차트 렌더링 기능을 검증하기 위한 테스트 파일입니다.

## 플로우차트 예시

```mermaid
graph TD
    A[시작] --> B{계속할까요?}
    B -->|예| C[작업 실행]
    B -->|아니오| D[종료]
    C --> E[결과 확인]
    E --> F{성공했나요?}
    F -->|예| G[완료]
    F -->|아니오| H[재시도]
    H --> C
    G --> D
```

## 시퀀스 다이어그램 예시

```mermaid
sequenceDiagram
    participant 사용자
    participant 시스템
    participant 데이터베이스

    사용자->>시스템: 요청 제출
    시스템->>데이터베이스: 데이터 조회
    데이터베이스-->>시스템: 결과 반환
    시스템-->>사용자: 결과 표시
```

## 간트 차트 예시

```mermaid
gantt
    title 프로젝트 일정
    dateFormat  YYYY-MM-DD
    section 설계
    요구사항 분석     :done,    des1, 2024-01-01, 7d
    프로토타입 설계     :active,  des2, 2024-01-08, 5d
    상세 설계     :         des3, after des2, 5d
    section 개발
    프론트엔드 개발     :         dev1, after des3, 10d
    백엔드 개발     :         dev2, after des3, 15d
    section 테스트
    단위 테스트     :         test1, after dev1, 5d
    통합 테스트     :         test2, after dev2, 5d
```

## 클래스 다이어그램 예시

```mermaid
classDiagram
    class Animal {
        +String name
        +int age
        +void eat()
        +void sleep()
    }
    class Dog {
        +void bark()
    }
    class Cat {
        +void meow()
    }

    Animal <|-- Dog
    Animal <|-- Cat
```

## 상태 다이어그램 예시

```mermaid
stateDiagram-v2
    [*] --> 대기
    대기 --> 처리중 : 처리 시작
    처리중 --> 성공 : 처리 성공
    처리중 --> 오류 : 처리 실패
    성공 --> [*]
    오류 --> [*]
```

## 파이 차트 예시

```mermaid
pie title 브라우저 사용률
    "Chrome" : 65.2
    "Firefox" : 15.3
    "Safari" : 12.5
    "Edge" : 5.2
    "기타" : 1.8
```

## 오류 구문 테스트 (오류 메시지가 표시되어야 함)

```mermaid
graph TD
    A --> B
    // 여기에 화살표 정의가 누락됨
    C --> D
```

이 테스트 파일은 CZON의 Mermaid 통합이 정상적으로 작동하는지 검증하기 위해 다양한 Mermaid 차트 유형을 포함하고 있습니다.

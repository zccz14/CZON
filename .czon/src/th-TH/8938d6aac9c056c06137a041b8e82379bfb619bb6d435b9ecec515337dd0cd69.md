# การทดสอบแผนภูมิ Mermaid

นี่คือไฟล์ทดสอบสำหรับตรวจสอบความสามารถในการแสดงผลแผนภูมิ Mermaid ใน CZON

## ตัวอย่างแผนผังลำดับงาน

```mermaid
graph TD
    A[เริ่มต้น] --> B{ดำเนินการต่อ?}
    B -->|ใช่| C[ดำเนินการ]
    B -->|ไม่| D[สิ้นสุด]
    C --> E[ตรวจสอบผลลัพธ์]
    E --> F{สำเร็จ?}
    F -->|ใช่| G[เสร็จสิ้น]
    F -->|ไม่| H[ลองใหม่]
    H --> C
    G --> D
```

## ตัวอย่างแผนผังลำดับเวลา

```mermaid
sequenceDiagram
    participant ผู้ใช้
    participant ระบบ
    participant ฐานข้อมูล

    ผู้ใช้->>ระบบ: ส่งคำขอ
    ระบบ->>ฐานข้อมูล: ค้นหาข้อมูล
    ฐานข้อมูล-->>ระบบ: ส่งกลับผลลัพธ์
    ระบบ-->>ผู้ใช้: แสดงผลลัพธ์
```

## ตัวอย่างแผนภูมิแกนต์

```mermaid
gantt
    title ตารางเวลาโครงการ
    dateFormat  YYYY-MM-DD
    section การออกแบบ
    วิเคราะห์ความต้องการ     :done,    des1, 2024-01-01, 7d
    ออกแบบต้นแบบ     :active,  des2, 2024-01-08, 5d
    ออกแบบรายละเอียด     :         des3, after des2, 5d
    section การพัฒนา
    พัฒนาด้านหน้าบ้าน     :         dev1, after des3, 10d
    พัฒนาด้านหลังบ้าน     :         dev2, after des3, 15d
    section การทดสอบ
    ทดสอบหน่วย     :         test1, after dev1, 5d
    ทดสอบการผสานรวม     :         test2, after dev2, 5d
```

## ตัวอย่างแผนภูมิคลาส

```mermaid
classDiagram
    class สัตว์ {
        +String ชื่อ
        +int อายุ
        +void กิน()
        +void นอน()
    }
    class สุนัข {
        +void เห่า()
    }
    class แมว {
        +void ร้อง()
    }

    สัตว์ <|-- สุนัข
    สัตว์ <|-- แมว
```

## ตัวอย่างแผนภูมิสถานะ

```mermaid
stateDiagram-v2
    [*] --> รออยู่
    รออยู่ --> กำลังประมวลผล : เริ่มประมวลผล
    กำลังประมวลผล --> สำเร็จ : ประมวลผลสำเร็จ
    กำลังประมวลผล --> ผิดพลาด : ประมวลผลล้มเหลว
    สำเร็จ --> [*]
    ผิดพลาด --> [*]
```

## ตัวอย่างแผนภูมิวงกลม

```mermaid
pie title อัตราการใช้เบราว์เซอร์
    "Chrome" : 65.2
    "Firefox" : 15.3
    "Safari" : 12.5
    "Edge" : 5.2
    "อื่นๆ" : 1.8
```

## การทดสอบไวยากรณ์ผิดพลาด (ควรแสดงข้อความผิดพลาด)

```mermaid
graph TD
    A --> B
    // ส่วนนี้ขาดคำจำกัดความลูกศร
    C --> D
```

ไฟล์ทดสอบนี้ประกอบด้วยแผนภูมิ Mermaid หลายประเภท เพื่อตรวจสอบว่าการผสานรวม Mermaid ใน CZON ทำงานได้อย่างถูกต้อง

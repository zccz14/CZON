# Astuces d'utilisation

## Déclarer la date dans le corps du texte

Dans le corps du texte, vous pouvez utiliser une déclaration de date telle que "Aujourd'hui, c'est le 8 janvier 2026" (le format est libre). CZON la reconnaîtra et la traitera automatiquement.

Les documents Markdown traditionnels déclarent la date via le YAML Frontmatter, ce qui nécessite de se souvenir du champ correspondant, sans quoi la date risque de ne pas être correctement identifiée.

CZON prend en charge la déclaration directe de la date dans le corps du texte, évitant ainsi les problèmes liés à l'oubli de remplir les champs Frontmatter.

L'IA extrait et déduit automatiquement les informations de date présentes dans le corps du texte et les utilise comme date de création du document.

Cette méthode est plus flexible et intelligente, améliorant ainsi l'expérience utilisateur.

À noter que les documents comportant une date peuvent générer une vue chronologique, facilitant la navigation et la gestion. Cela convient particulièrement aux types de contenu nécessitant une chronologie, tels que les journaux personnels ou les blogs.

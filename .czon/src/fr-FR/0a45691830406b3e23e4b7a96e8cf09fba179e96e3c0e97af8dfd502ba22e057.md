# Astuces d'utilisation

## Déclarer une date dans le corps du texte

Dans le corps du texte, vous pouvez utiliser une déclaration de date comme "Aujourd'hui, nous sommes le 8 janvier 2026" (le format et le style sont libres). CZON la reconnaîtra et la traitera automatiquement.

Traditionnellement, les documents Markdown déclarent la date via le YAML Frontmatter, ce qui nécessite de se souvenir du champ correspondant, sans quoi la date risque de ne pas être correctement identifiée.

CZON prend en charge la déclaration directe de dates dans le corps du texte, évitant ainsi les problèmes d'oubli de remplissage des champs du Frontmatter.

L'IA extrait et déduit automatiquement les informations de date présentes dans le texte et les traite comme date de création du document.

Cette méthode est plus flexible et intelligente, améliorant ainsi l'expérience utilisateur.

Au passage, les documents comportant une date peuvent générer une vue chronologique, facilitant la navigation et la gestion. C'est particulièrement adapté aux journaux, blogs et autres types de contenu nécessitant une ligne du temps.

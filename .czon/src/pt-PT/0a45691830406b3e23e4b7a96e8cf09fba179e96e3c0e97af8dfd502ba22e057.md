# Técnicas de Utilização

## Declarar a Data no Corpo do Texto

No corpo do texto, pode utilizar declarações de data como "Hoje é 8 de janeiro de 2026" (o estilo do formato é livre). O CZON irá reconhecer e processar automaticamente.

Os documentos Markdown tradicionais declaram a data através do YAML Frontmatter, o que exige a memorização do campo correspondente; caso contrário, a data pode não ser corretamente identificada.

O CZON suporta a declaração direta da data no corpo do texto, evitando o problema de esquecer o preenchimento do campo no Frontmatter.

A IA extrairá e inferirá automaticamente a informação da data presente no corpo do texto, tratando-a como a data de criação do documento.

Esta abordagem é mais flexível e inteligente, melhorando a experiência do utilizador.

A propósito, documentos que possuem uma data podem gerar uma vista de linha do tempo, facilitando a navegação e gestão. É especialmente adequado para tipos de conteúdo como diários, blogs ou outros que necessitem de uma linha do tempo.

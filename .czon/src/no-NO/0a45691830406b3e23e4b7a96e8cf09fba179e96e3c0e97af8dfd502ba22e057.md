# Bruksteknikker

## Erklær dato i hovedteksten

I hovedteksten kan du bruke datoerklæringer som "i dag er 8. januar 2026" (fri formateringsstil), og CZON vil automatisk gjenkjenne og behandle disse.

Tradisjonelle Markdown-dokumenter erklærer dato via YAML Frontmatter, noe som krever at man husker de riktige feltene, ellers kan datoen ikke gjenkjennes korrekt.

CZON støtter direkte datoerklæring i hovedteksten, noe som unngår problemet med å glemme å fylle ut Frontmatter-feltene.

AI vil automatisk trekke ut og utlede datoinformasjonen fra hovedteksten og behandle den som dokumentets opprettelsesdato.

Denne metoden er mer fleksibel og intelligent og forbedrer brukeropplevelsen.

Forresten, dokumenter med datoer kan generere tidslinje-visninger, noe som gjør det enkelt å bla gjennom og administrere. Dette er spesielt egnet for innholdstyper som dagbøker, blogger og annet som krever tidslinjer.

# Användningstekniker

## Deklarera datum i brödtexten

I brödtexten kan man använda datumdeklarationer som "Idag är den 8 januari 2026" (formatstil är fri), och CZON kommer automatiskt att känna igen och behandla detta.

Traditionella Markdown-dokument deklarerar datum via YAML Frontmatter, vilket kräver att man kommer ihåg motsvarande fält, annars kan datumet inte identifieras korrekt.

CZON stöder direkt datumdeklaration i brödtexten, vilket undviker problemet med att glömma att fylla i Frontmatter-fält.

AI kommer automatiskt att extrahera och härleda datuminformation från brödtexten och behandla det som dokumentets skapandedatum.

Denna metod är mer flexibel och intelligent, vilket förbättrar användarupplevelsen.

För övrigt kan dokument med datum generera tidslinjevyer för enkel navigering och hantering. Särskilt lämpligt för innehållstyper som dagböcker, bloggar etc. som behöver tidslinjer.

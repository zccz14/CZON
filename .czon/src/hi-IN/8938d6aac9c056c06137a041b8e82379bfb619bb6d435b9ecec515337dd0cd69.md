# Mermaid चार्ट परीक्षण

यह एक परीक्षण फ़ाइल है जो CZON में Mermaid चार्ट रेंडरिंग कार्यक्षमता को सत्यापित करने के लिए है।

## फ़्लोचार्ट उदाहरण

```mermaid
graph TD
    A[शुरू] --> B{जारी रखें?}
    B -->|हाँ| C[क्रिया निष्पादित करें]
    B -->|नहीं| D[समाप्त]
    C --> E[परिणाम जाँचें]
    E --> F{सफल?}
    F -->|हाँ| G[पूर्ण]
    F -->|नहीं| H[पुनः प्रयास करें]
    H --> C
    G --> D
```

## सीक्वेंस डायग्राम उदाहरण

```mermaid
sequenceDiagram
    participant उपयोगकर्ता
    participant सिस्टम
    participant डेटाबेस

    उपयोगकर्ता->>सिस्टम: अनुरोध सबमिट करें
    सिस्टम->>डेटाबेस: डेटा क्वेरी करें
    डेटाबेस-->>सिस्टम: परिणाम वापस करें
    सिस्टम-->>उपयोगकर्ता: परिणाम दिखाएँ
```

## गैंट चार्ट उदाहरण

```mermaid
gantt
    title परियोजना समयसारणी
    dateFormat  YYYY-MM-DD
    section डिज़ाइन
    आवश्यकता विश्लेषण     :done,    des1, 2024-01-01, 7d
    प्रोटोटाइप डिज़ाइन     :active,  des2, 2024-01-08, 5d
    विस्तृत डिज़ाइन     :         des3, after des2, 5d
    section विकास
    फ्रंटएंड विकास     :         dev1, after des3, 10d
    बैकएंड विकास     :         dev2, after des3, 15d
    section परीक्षण
    यूनिट परीक्षण     :         test1, after dev1, 5d
    एकीकरण परीक्षण     :         test2, after dev2, 5d
```

## क्लास डायग्राम उदाहरण

```mermaid
classDiagram
    class Animal {
        +String name
        +int age
        +void eat()
        +void sleep()
    }
    class Dog {
        +void bark()
    }
    class Cat {
        +void meow()
    }

    Animal <|-- Dog
    Animal <|-- Cat
```

```mermaid
stateDiagram-v2

    निष्क्रिय --> प्रसंस्करण : प्रसंस्करण शुरू करें

    प्रसंस्करण --> त्रुटि : प्रसंस्करण विफल

    त्रुटि --> [*]
```

## त्रुटिपूर्ण सिंटैक्स परीक्षण (त्रुटि संदेश दिखाना चाहिए)

```mermaid
graph TD
    A --> B
    // यहाँ तीर परिभाषा गायब है
    C --> D
```

इस परीक्षण फ़ाइल में विभिन्न प्रकार के Mermaid चार्ट शामिल हैं, जिनका उपयोग CZON के Mermaid एकीकरण के सही ढंग से काम करने की पुष्टि करने के लिए किया जाता है।

# Mermaid चार्ट परीक्षण

यह CZON में Mermaid चार्ट रेंडरिंग कार्यक्षमता को सत्यापित करने के लिए एक परीक्षण फ़ाइल है।

## फ़्लोचार्ट उदाहरण

```mermaid
graph TD
    A[शुरू] --> B{क्या जारी रखें?}
    B -->|हाँ| C[क्रिया निष्पादित करें]
    B -->|नहीं| D[समाप्त]
    C --> E[परिणाम जाँचें]
    E --> F{क्या सफल?}
    F -->|हाँ| G[पूर्ण]
    F -->|नहीं| H[पुनः प्रयास करें]
    H --> C
    G --> D
```

## सीक्वेंस डायग्राम उदाहरण

```mermaid
sequenceDiagram
    participant उपयोगकर्ता
    participant सिस्टम
    participant डेटाबेस

    उपयोगकर्ता->>सिस्टम: अनुरोध सबमिट करें
    सिस्टम->>डेटाबेस: डेटा क्वेरी करें
    डेटाबेस-->>सिस्टम: परिणाम लौटाएँ
    सिस्टम-->>उपयोगकर्ता: परिणाम दिखाएँ
```

## गैंट चार्ट उदाहरण

```mermaid
gantt
    title परियोजना समयसारणी
    dateFormat  YYYY-MM-DD
    section डिज़ाइन
    आवश्यकता विश्लेषण     :पूर्ण,    des1, 2024-01-01, 7d
    प्रोटोटाइप डिज़ाइन     :सक्रिय,  des2, 2024-01-08, 5d
    विस्तृत डिज़ाइन     :         des3, des2 के बाद, 5d
    section विकास
    फ्रंटएंड विकास     :         dev1, des3 के बाद, 10d
    बैकएंड विकास     :         dev2, des3 के बाद, 15d
    section परीक्षण
    यूनिट परीक्षण     :         test1, dev1 के बाद, 5d
    एकीकरण परीक्षण     :         test2, dev2 के बाद, 5d
```

## क्लास डायग्राम उदाहरण

```mermaid
classDiagram
    class Animal {
        +String name
        +int age
        +void eat()
        +void sleep()
    }
    class Dog {
        +void bark()
    }
    class Cat {
        +void meow()
    }

    Animal <|-- Dog
    Animal <|-- Cat
```

## स्टेट डायग्राम उदाहरण

```mermaid
stateDiagram-v2
    [*] --> निष्क्रिय
    निष्क्रिय --> प्रसंस्करण : प्रसंस्करण शुरू करें
    प्रसंस्करण --> सफलता : प्रसंस्करण सफल
    प्रसंस्करण --> त्रुटि : प्रसंस्करण विफल
    सफलता --> [*]
    त्रुटि --> [*]
```

## त्रुटिपूर्ण वाक्यविन्यास परीक्षण (त्रुटि संदेश दिखना चाहिए)

```mermaid
graph TD
    A --> B
    // यहाँ तीर परिभाषा गायब है

```

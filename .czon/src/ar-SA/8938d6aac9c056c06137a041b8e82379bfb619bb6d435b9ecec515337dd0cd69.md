# اختبار مخططات Mermaid

هذا ملف اختبار للتحقق من وظيفة عرض مخططات Mermaid في CZON.

## مثال على المخطط الانسيابي

```mermaid
graph TD
    A[بداية] --> B{هل تستمر؟}
    B -->|نعم| C[تنفيذ العملية]
    B -->|لا| D[نهاية]
    C --> E[فحص النتيجة]
    E --> F{هل نجحت؟}
    F -->|نعم| G[اكتمال]
    F -->|لا| H[إعادة المحاولة]
    H --> C
    G --> D
```

## مثال على مخطط التسلسل

```mermaid
sequenceDiagram
    participant المستخدم
    participant النظام
    participant قاعدة_البيانات

    المستخدم->>النظام: تقديم الطلب
    النظام->>قاعدة_البيانات: استعلام البيانات
    قاعدة_البيانات-->>النظام: إرجاع النتيجة
    النظام-->>المستخدم: عرض النتيجة
```

## مثال على مخطط جانت

```mermaid
gantt
    title الجدول الزمني للمشروع
    dateFormat  YYYY-MM-DD
    section التصميم
    تحليل المتطلبات     :منتهي,    des1, 2024-01-01, 7d
    تصميم النموذج الأولي     :نشط,  des2, 2024-01-08, 5d
    التصميم التفصيلي     :         des3, after des2, 5d
    section التطوير
    تطوير الواجهة الأمامية     :         dev1, after des3, 10d
    تطوير الواجهة الخلفية     :         dev2, after des3, 15d
    section الاختبار
    اختبار الوحدات     :         test1, after dev1, 5d
    اختبار التكامل     :         test2, after dev2, 5d
```

## مثال على مخطط الفئات

```mermaid
classDiagram
    class الحيوان {
        +String الاسم
        +int العمر
        +void أكل()
        +void نوم()
    }
    class الكلب {
        +void نباح()
    }
    class القط {
        +void مواء()
    }

    الحيوان <|-- الكلب
    الحيوان <|-- القط
```

## مثال على مخطط الحالة

```mermaid
stateDiagram-v2
    [*] --> خامل
    خامل --> معالجة : بدء المعالجة
    معالجة --> نجاح : نجاح المعالجة
    معالجة --> خطأ : فشل المعالجة
    نجاح --> [*]
    خطأ --> [*]
```

## مثال على المخطط الدائري

```mermaid
pie title معدل استخدام المتصفحات
    "Chrome" : 65.2
    "Firefox" : 15.3
    "Safari" : 12.5
    "Edge" : 5.2
    "أخرى" : 1.8
```

## اختبار بناء الجملة الخاطئ (يجب أن يعرض رسالة خطأ)

```mermaid
graph TD
    A --> B
    // هنا تعريف السهم مفقود
    C --> D
```

يحتوي ملف الاختبار هذا على أنواع متعددة من مخططات Mermaid للتحقق مما إذا كان تكامل CZON مع Mermaid يعمل بشكل صحيح.

# Técnicas de Uso

## Declarando a Data no Corpo do Texto

No corpo do texto, você pode usar declarações de data como "Hoje é 8 de janeiro de 2026" (o estilo de formatação é livre). O CZON irá reconhecer e processar automaticamente.

Documentos Markdown tradicionais declaram a data através do YAML Frontmatter, exigindo que o usuário lembre do campo correspondente; caso contrário, a data pode não ser identificada corretamente.

O CZON suporta a declaração direta da data no corpo do texto, evitando o problema de esquecer de preencher os campos do Frontmatter.

A IA extrai e infere automaticamente as informações de data no corpo do texto e as processa como a data de criação do documento.

Essa abordagem é mais flexível e inteligente, melhorando a experiência do usuário.

A propósito, documentos com datas podem gerar visualizações de linha do tempo, facilitando a navegação e o gerenciamento. É especialmente adequado para tipos de conteúdo que exigem uma linha do tempo, como diários e blogs.
